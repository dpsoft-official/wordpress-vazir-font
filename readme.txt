فونت وزیر برای وردپرس

== Description ==
**فونت وزیر برای داشبورد و قالب وردپرس**

 با این افزونه می توانید فونت وزیر را برای داشبورد و قالب وردپرس تنظیم کنید.

= امکانات =
 * پنل تنظیمات ساده
 * قابلیت تنظیم CDN فونت
 * قابلیت تنظیم نام فونت وزیر
* قابل استفاده در قالب های پیش فرض پردپرس
* قابل استفاده در قالب های پر استفاده وردپرس مانند neve
== Installation ==
2. پوشه `vazir-font` را در مسیر `/wp-content/plugins/` آپلود کنید
3. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید
4. تنظیمات افزونه را می توانید از طریق قسمت تنظیمات / وزیر فونت انجام دهید
